# PuttMasters - Premium Mini Golf & Putting Course HTML5 Template

A commercial-grade, premium family entertainment and tourist attraction static website template. 

## Technical Highlights
- **HTML5 & CSS3**: Native custom properties with dynamic, flicker-free Light & Dark theme settings.
- **Grid Framework**: Local Bootstrap 5 Grid for responsive column control.
- **Animations**: GSAP & ScrollTrigger fade-ups and stagger effects.
- **Asset caching**: All featured image assets downloaded locally.

## File Structure
- `/` - 13 static pages (e.g. index.html, home-2.html, courses.html, course-details.html).
- `/assets/css/` - Custom styles, layout definitions, dark theme variables, and keyframe animations.
- `/assets/js/` - General responsive interactions, theme toggler logic, and GSAP scroll indicators.
- `/assets/images/` - Cached Unsplash landscape photography assets organized in categories.

## Setup & Running
No compilation steps are needed. Double-click `index.html` to load the application directly in any browser, or host via simple local servers (e.g. `npx serve` or Live Server extension).
